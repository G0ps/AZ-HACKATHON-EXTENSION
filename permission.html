Chrome Permission Instructions

If you're still having issues with microphone permissions in Chrome, follow these steps:

  1. Click on the lock icon (or site settings icon) in the address bar
  2. Make sure "Microphone" is set to "Allow"
  3. If you don't see the microphone option or it's blocked, you may need to reset permissions for this site
  4. You can also go to Chrome Settings → Privacy and Security → Site Settings → Microphone to manage permissions

After granting permission, return to the extension popup and try again.
